# Zomato Data Analysis using Python

## Project Overview
This project analyzes Zomato restaurant data using Python to identify customer behavior, restaurant ratings, and online ordering trends using data visualization techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Features
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Correlation Analysis
